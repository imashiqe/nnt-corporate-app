# NNT-style Bootstrap Website Clone

This is a static, responsive front-end clone inspired by the provided NNT website screenshots.

## Included pages

- `index.html` - home page with hero Swiper slider, clients, services, product types, gallery, testimonials, videos, blogs and footer
- `about.html` - about, mission, vision, leader profile and legal papers
- `services.html` - core services and industry segments
- `custom-bag.html` - quotation section, quote form, packages, gallery and factory tour
- `blog.html` - blog grid with sidebar
- `blog-detail.html` - article detail with table and sidebar

## Libraries used from CDN

- Bootstrap 5
- jQuery 3
- AOS animation
- Font Awesome 6
- Swiper slider
- Google Fonts

## How to run

Open `index.html` in a browser or upload the folder to any static hosting service. Because CDN libraries are used, an internet connection is required for Bootstrap, jQuery, AOS, Font Awesome, Swiper and Google Fonts.

## Important notes

- The included images are locally generated SVG placeholders, not downloaded from the original site.
- Replace logo, text, phone, email, address and images with your licensed brand assets before production.
- The quote form is front-end only. Connect it to your backend, WordPress, PHP mailer, API endpoint or CRM to receive submissions.
